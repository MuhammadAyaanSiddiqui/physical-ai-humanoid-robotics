## Image Specifications

- **Format**: PNG (diagrams), JPG (photos)
- **Resolution**: Minimum 1200px width for detailed diagrams
- **File Size**: <500KB per image (optimize for web)
- **Alt Text**: All images should have descriptive alt text for accessibility

## Generating Images

Use tools like:
- **Diagrams**: Mermaid, draw.io, Lucidchart
- **Screenshots**: Capture from Isaac Sim, ROS 2 tools, API responses
- **Visualizations**: Matplotlib, Plotly for graphs and charts
